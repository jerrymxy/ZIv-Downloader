# ZIv-Downloader

Download simfiles pack from ZIv / Zenius-I-vanisher.com

## Usage

```
java -jar ZIv-Downloader.jar [url] [path]
```

Or just run the .jar file and follow the tips.

## Requirements

Java >= 11
